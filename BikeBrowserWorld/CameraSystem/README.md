# Camera System

Copy this folder into your Godot project and add `CameraController.gd` as an autoload if you want global calls such as `CameraController.shake(5.0, 0.3)`.

Use `Camera2DPlus.gd` on the active `Camera2D` for smooth follow, look-ahead, dead zones, drag margins, zoom limits, and world edge limits through Godot's built-in camera limit properties.
