# Prop Generation Prompt Sets

Use these prompts one category at a time. Generate separate PNG files for each listed prop.

Global rules for every prompt:

- Style: pixel art, GBA/SNES-inspired.
- Palette: warm Sonoran dusk colors.
- Background: transparent.
- Format: PNG.
- Save to the exact category folder listed.
- Preserve raw generated atlases if an atlas workflow is used.
- Create a contact sheet for each category after splitting so assets can be reviewed quickly.
- Do not replace existing files unless intentionally regenerating a better version.
