## Sprite Sheet Requirements

For any generated animation:

- Use transparent background.
- Fixed frame size.
- Even spacing.
- No shadows cropped off.
- No text.
- Include frame count in filename.
- Save raw and cleaned versions separately.

Recommended naming:

- `bmx_repair_stand_slipped_chain.png`
- `bmx_repair_stand_aligned_chain.png`
- `bmx_repair_stand_fixed_chain.png`
